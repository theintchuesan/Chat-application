# 💬 Chat Application (Java + HTML)

A simple real-time chat application built using **Java (backend)** and **HTML/CSS/JavaScript (frontend)**.

## 🚀 Features

* Real-time messaging
* Multiple users support
* Simple chat UI
* Message broadcast to all users
* Lightweight and easy to run

## 🛠️ Tech Stack

* Java (Server-side / Backend)
* HTML (UI structure)
* CSS (Styling)
* JavaScript (Client-side logic)
* WebSocket / Socket Programming (for real-time communication)

## ▶️ How to Run

1. Start the Java server (run main server file)
2. Open `index.html` in a browser
3. Connect multiple users to start chatting

## 📂 Project Structure

* `index.html` – chat UI
* `style.css` – styling
* `script.js` – client logic
* `ChatServer.java` – server-side logic
* `ClientHandler.java` – manages user connections
